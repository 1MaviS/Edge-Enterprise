# Release notes for Microsoft Edge Security Updates

These release notes provide information about security fixes that are included in updates to Microsoft Edge Stable channel.

Microsoft has released the latest Microsoft Edge Stable Channel (Version 91.0.864.54), which incorporates the latest Security Updates of the Chromium project. For more information, see the [Security Update Guide](https://msrc.microsoft.com/update-guide)
